Portfolio



Holiday Wishlist

Save and Share favorite items from the web using Etsy API.

Holiday Wishlist is a wishlist application that lets you save your favorite items on the internet or through Etsy API. It lets you share your wishlist with your friends and let you view their wishlist--making Secret Santa easier than it has ever been before! Users can create a wishlist, save items from Etsy on their wishlist and view their friends' wishlist. Future goals include incorporating the WePay API so users can purchase items on their own wishlist or for their friends.

Technlogies used: Devise for user authentication, Etsy API, Javascript for Snowfall, Masonary JS for nice grid layout, Bootstrap for responsiveness.

Fitbud

FitBud makes it easier than ever find a new workout buddy.

Fitbud is an application that makes it easy for users to find others interested in their favorite outdoor activity. Finding the motivation to workout alone can be difficult, but with the help of FitBud, you can make a new friend to help you stay motivated and on track with your goals! Users can sign up, pick favorite activity, and find and message others who also chose a similar activity.

Techonologies Used: Devise for user authentication, Paperclip for profile picture upload, Foundation for responsiveness, HTML/SCSS.

Indian Food Finder

Shiftable

Shift change requests have never been this fun!

Shiftable was made as a group project built on Ruby on Rails & JavaScript. Shiftable provides a reliable, convenient, and simple platform for employees and shift-managers to manage, share and edit shift schedules. Users can log in, admin can assign work shifts to employees, employees can request shift change and admin has the ability to approve or reject requests.

Technlogies used: Devise for user authentication, Full Calendar JQuery Plugin, Javascript and jQuery, Action Mailer.
